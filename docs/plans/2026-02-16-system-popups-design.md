# 시스템 안내 팝업 디자인

## 개요

System 섹션의 4개 카드를 클릭하면 풀스크린 오버레이 모달이 열려 각 시스템에 대한 중간 상세 수준의 안내를 보여준다. 모달 하단에 노션 상세 문서 링크를 연결한다.

## 시스템 카드 구성 (변경)

| 코드 | 라벨          | 변경 사항                  |
| ---- | ------------- | -------------------------- |
| GM   | AI 전투 판정  | 유지                       |
| SYNC | 공명율 & 스킬 | 유지                       |
| ARC  | 시즌 스토리   | 유지                       |
| OC   | 캐릭터 생성   | **신규** (기존 DREAM 대체) |

DREAM(꿈 메카닉)은 World 섹션에서 이미 다루고 있으므로 System에서 제거.

## 모달 디자인

### 레이아웃

```
┌─────────────────────────────────────────┐
│ ┌─ HUD BRACKET                  [X] ─┐ │
│ │                                      ││
│ │  [GLYPH]  시스템 타이틀              ││
│ │  ─────────────────────               ││
│ │                                      ││
│ │  (콘텐츠 영역)                        ││
│ │                                      ││
│ │  ─────────────────────               ││
│ │  📎 상세 문서 보기 →  (노션 링크)     ││
│ │                                      ││
│ └─                                   ─┘│
└─────────────────────────────────────────┘
     (배경: blur + dark overlay)
```

### 비주얼 스펙

- **배경 오버레이:** `bg-black/80` + `backdrop-filter: blur(8px)`
- **모달 프레임:** HUD 코너 브래킷 (시안), `max-w-[640px]`, 중앙 정렬
- **헤더:** 시스템 글리프 원형 배지 + 타이틀 (기존 카드 패턴 재사용)
- **닫기 버튼:** 우측 상단, HUD 브래킷 안쪽
- **하단 링크:** 노션 상세 문서로 연결

### 닫기 인터랙션

- ESC 키
- 배경(오버레이) 클릭
- X 버튼 클릭

### 열림 애니메이션

1. 오버레이 페이드인 (0.2s)
2. 스캔라인 스윕 (0.3s)
3. HUD 브래킷 확장 (0.3s)
4. 콘텐츠 페이드인 (0.3s)

### 닫힘 애니메이션

- 콘텐츠 + 프레임 페이드아웃 (0.2s)
- 오버레이 페이드아웃 (0.2s)

## 기술 구조

### 접근 방식: 단일 모달 컴포넌트

하나의 `SystemModal.tsx`가 모든 시스템 팝업을 처리. 데이터만 교체.

### 파일 변경

- `System.tsx` — 카드 데이터 변경 (DREAM→OC), 클릭 핸들러 추가
- `SystemModal.tsx` (신규) — 풀스크린 오버레이 모달 컴포넌트
- `systemData.ts` (신규) — 4개 시스템별 콘텐츠 데이터

### 콘텐츠 데이터 구조

```typescript
interface SystemInfo {
  code: string; // "GM" | "SYNC" | "ARC" | "OC"
  title: string; // 시스템 타이틀
  glyph: string; // 글리프 텍스트
  description: string; // 카드 위 짧은 설명
  sections: {
    heading: string;
    body: string;
  }[];
  notionUrl?: string; // 노션 상세 문서 링크
}
```

### 콘텐츠

플레이스홀더로 구현. 실제 텍스트는 추후 사용자가 지시.

## 반응형

- 데스크탑: 640px 모달, 중앙 정렬
- 모바일: 전체 너비, 패딩 축소, 스크롤 가능

## 접근성

- ESC로 닫기
- 모달 열릴 때 body scroll 잠금
- focus trap (모달 내부에서만 탭 이동)
