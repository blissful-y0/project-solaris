- **타이핑 애니메이션:**

  2. "공명율 측정을 시작합니다" (1초 후)

  - **"나는 질서를 지킨다"** → "공명율: 87% — 보안국 적합 판정" (앰버 글로우)
  - **"나는 진실을 찾는다"** → "공명율: 12% — 추방 대상" (레드 글로우)
  - **"나는 아직 모른다"** → "공명율: 측정 불가 — 관찰 대상" (회색)
- 선택 후 결과 텍스트 + 아래로 스크롤 유도 화살표
- **모바일:** 카드 세로 스택, 터치 친화적 크기
