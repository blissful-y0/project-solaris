# 세계관 문서 인덱스
이 디렉토리는 Notion 원본 세계관 문서를 플레이어/운영/AI GM 관점으로 재구성한 통합 세계관 기준서다.

## 문서 맵
### 세계관 기초
- `world-overview.md`: 솔라리스의 시대 배경, 돔 도시 구조, 외부 황무지와 꿈의 의미
- `helios-core.md`: 헬리오스의 정체, 감정 수확 체계, 통제 메커니즘과 비밀 권력 구조
- `society.md`: 화폐 없는 배급 사회의 운영 논리, 계층, 시민 통제 서사

### 캐릭터/전투 기초
- `resonance-and-powers.md`: 공명율(Sync Rate) 체계, 각성 조건, 전향자/비능력자 구조
- `combat-system.md`: 능력 유형/계열, 판정 원칙, 전투 흐름, 코스트와 상태 규칙
- `factions.md`: 보안국(SBCS/SDF), 스태틱(The Static), 시민, 전향자의 목적과 충돌 축

### 운영/서사 시스템
- `operations.md`: 커뮤니티 운영 규칙, 시즌 운영, 참가/활동 규약, 캐릭터 생성 가이드
- `ai-gm-system.md`: AI GM 데이터 모델, 컨텍스트 최적화, 환각 방지, 아카이빙 전략

## 용어 사전 (Glossary)
### 세계 구조
- 헬리오스 코어(Helios Core): 도시 에너지/통제를 총괄하는 자아형 AI
- 공명판(Resonance Plate): 감정 파장 수집 및 REM 억제 인프라
- 감정 잔향(Emotional Residue): 감정 수확 후 재분배되는 피드백 파장

### 인물/집단
- 보안국(Solaris Bureau of Civic Security, SBCS): 헬리오스 질서 집행 세력
- 태양방위군(Solar Defense Force, SDF): 보안국 전투 집행 조직 표기
- 스태틱(The Static): 추방자/이탈자 기반 저항 세력
- 전향자(Defector): 보안국에서 스태틱으로 이탈한 고공명 능력자

### 규약/시스템
- 공명율(Sync Rate/Resonance Rate): 헬리오스와의 동기화 수치
- 공명 측정(Resonance Calibration): 15세 확인식
- 석양 의정서(Sunset Protocol): 기준 미달자의 추방 절차
- 하모닉스 프로토콜(Harmonics Protocol): 보안국형 능력 운용 방식(WILL 소모)
- 오버드라이브(Overdrive): 스태틱형 능력 운용 방식(HP 소모)
